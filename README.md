# Poké Role - Pokédex

A Foundry VTT v13 module for the [Pok-Role-Module](https://github.com/RiccardoMont1/Pok-Role-Module) system.

Adds an animated Pokédex button below the Token Controls button in the left scene-controls column. Target a Pokémon token (press `T`) then click the Pokédex button to open a modern clamshell Pokédex overlay showing:

- Species portrait (actor image)
- Type(s) with coloured chips and type icons
- Pokédex entry / biography

## Compatibility

- Foundry VTT **v13** (tested on build 351)
- System: **pok-role-system** (Poké Role)

## Usage

1. Select and target a Pokémon token.
2. Click the **Pokédex** button in the left scene-controls column (right under the Tokens button).
3. Close the Pokédex with the red/blue button, by pressing `Esc`, or by clicking outside the device.
